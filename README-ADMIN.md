##  Email Setup

Quiz results are automatically sent to your email via FormSubmit.

**To update email:**
1. Open \script.js\
2. Find line ~1023:
   \\\javascript
   fetch('https://formsubmit.co/ajax/YOUR-EMAIL@example.com'
   \\\
3. Replace with your real email
4. Save and push to GitHub
